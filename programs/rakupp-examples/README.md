# rakupp-examples — the Raku++ cookbook (github.com/ash/rakupp, examples/)

24 small, self-contained idiomatic Raku programs (grammars, recursion, text
processing, an interpreter, a ray of classic algorithms). Snapshot taken
2026-07-22 from rakupp v1.0.0; references generated from Rakudo v2026.06 like
every other unit. Server/interactive programs classify as ETERNAL/NONDET and
sit outside the diffable set automatically.
